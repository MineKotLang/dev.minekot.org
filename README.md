# dev.minekot.org

*Developer documentation for MineKot*

## Why?

MineKot is a Kotlin-based framework designed to streamline Minecraft server development. This repository hosts the documentation and source for `dev.minekot.org`, providing developers with API references, best practices, and technical specifications for building on the MineKot ecosystem.

## Usage

You can browse the documentation locally by running a static site generator or visiting the live site. For contributors, the documentation is written in Markdown and organized by modules to ensure clarity and ease of navigation across the different components of the framework.
