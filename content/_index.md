Welcome to the MineKot Developer Documentation!

We are glad to see you're interested in contributing to MineKot. This site serves as the central hub for technical specifications, API references, and contribution guidelines for the MineKot ecosystem.

## Getting started

Whether you are looking to develop modules, contribute to the core engine, or improve our infrastructure, you will find the necessary resources here. Please ensure you read our coding standards and setup guides before submitting your first pull request.

Start with [coding standards and style]({{% relref "codestyle" %}}) for project-wide rules, or [MineKot Toolchain]({{% relref "toolchain" %}}) when working on the Gradle plugin, Detekt rules, generated project files, or shared helper libraries.
